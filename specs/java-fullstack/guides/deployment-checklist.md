# 部署检查清单

## 构建

- [ ] `mvn -B clean verify` 通过。
- [ ] 依赖漏洞和许可证检查通过。
- [ ] 构建产物来自 CI，不使用本地手工包。
- [ ] JDK、Maven、插件版本固定。

## 配置

- [ ] 生产配置来自安全配置源。
- [ ] 密钥未出现在仓库、日志和构建产物中。
- [ ] 数据库、缓存、消息、第三方服务地址正确。
- [ ] 超时、线程池、连接池配置已确认。

## 数据库

- [ ] 迁移脚本已在测试环境验证。
- [ ] 大表变更已评估锁表风险。
- [ ] 回滚或修复方案已准备。
- [ ] 应用版本与数据库结构兼容。

## 可观测性

- [ ] 健康检查正常。
- [ ] 日志、指标、链路追踪可用。
- [ ] 关键告警已配置。
- [ ] 新功能有业务指标或审计日志。
